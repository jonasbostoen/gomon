# Gomon: a network traffic analyzer written in Go

## Version 1
* Get all interfaces
* Listen on interfaces
* Unpack packets in corresponding structs

## Dependencies
### Linux
github.com/google/gopacket

### Windows
github.com/google/gopacket
golang.org/x/sys/windows
